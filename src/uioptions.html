About UI Options

User Interface Options (or UI Options) provides a way to enhance or improve website usability, flexibility, and accessibility by providing a way to customize and control aspects of a website without the need for additional software or tools. For those who design, build, or maintain websites, UI Options can also help reveal areas where the web content and structure can be changed to improve flexibility and robustness.

Get UI Options

There are a number of different ways to get UI Options for your project. Visit Infusion Docs | Getting User Interface Options to choose the best method for your situation.

Guides and Documentation

Infusion Docs | Setting Up User Interface Options is a guide that will help you get started with using User Interface Options in your project. Once you have UI Options installed in your project, Infusion Docs | Working with UI Options can help guide you through conifguring your site's styles and markup to take advantage of UI Options.
